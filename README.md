# Chess GUI

This is a GUI for chess implemented using Flask and UCI.

## Usage

```bash
docker-compose up -d
```

## Installation

```bash
git clone git@github.com:Bungogood/Chess-Gui.git
cd Chess-Gui
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
